Introduction

JavaScript is what "makes web pages alive."

The programs in this language is called scripts. They can be written right in a web page's HTML and run automatically as the page loads.

Variables

A variable is a "named storage" for data. We can use variables to store goodies, visitors, and other data.

Variables aren't the values themselves. That is important to note. They are the containers for the values.

let message;

message; = "Hello World";

alert(message);
Data Types

A value in JavaScript is always a certain type. For example, a string or a number.

There are eight basic data types in JavaScript. We can put any type into a variable:

  • String
  • Number
  • Boolean
  • Null
  • Undefined
  • Symbol
  • Object
  • BigInt
let message = "Hello" //variable is a string

message = 123456; //now it is a number

alert(message); //1234 will be alerted
Conditionals

Sometimes, we need to perform different actions based on different conditions.

These conditions can be in the form of an if...else statement or a switch statement.
